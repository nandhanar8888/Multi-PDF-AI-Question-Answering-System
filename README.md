# SmartPDF – Multi-PDF AI Question Answering System

A Flask-based AI application that allows users to upload multiple PDF documents and ask natural language questions. The system extracts text from PDFs, retrieves the most relevant content using TF-IDF, and generates answers using the Hugging Face Inference API.

## Features

- Upload and process multiple PDF documents
- Ask questions in natural language
- TF-IDF based document retrieval
- AI-powered responses using Hugging Face API
- Built with Flask and PyPDF2
- Interactive web interface

## Tech Stack

- Python
- Flask
- PyPDF2
- Scikit-learn (TF-IDF)
- Hugging Face Inference API
- HTML/CSS

## Installation

```bash
git clone https://github.com/nandhanar8888/SmartPDF-AI-Question-Answering-System.git
cd SmartPDF-AI-Question-Answering-System
pip install -r requirements.txt
python app.py
```

## Note

Uploaded PDF files and API keys are excluded from this repository for security and privacy.
